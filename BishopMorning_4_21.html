Quick Reference — Key Terms Used In This Report

Brief definitions for the positioning and structure language used throughout. Skip if already fluent.

GEX Flip
Price where dealer behavior inverts. Above = dealers absorb moves. Below = dealers amplify them.
Charm
Delta decay from time passing. In a call-heavy book, every day = dealers sell shares. No off switch.
Vanna
How delta changes when IV moves. Negative vanna (now) = vol expansion forces dealer selling.
OI Gamma
Open interest gamma — the persistent dealer positioning carried overnight. Currently NEGATIVE in SPY.
GEX Magnet
A strike with concentrated positive gamma that attracts price as dealers hedge around it.
Dealer Short
Shares dealers are short as a hedge for puts they sold. Unwinding this = forced buying pressure.
NCP / NPP
Net Call / Put Premium. Dollar-weighted flow into calls vs puts. More meaningful than simple volume.
0DTE / 1DTE
Days to expiration. Short-dated options dominate intraday dealer hedging — highest gamma per dollar.
Buyback Wall
Corporate repurchase programs. $428B authorized YTD. Blackout lifts ~2 weeks — ~$18B/week demand.

SPY OI Gamma — The Structural Break
SPY OI gamma is negative (−$1.86B per 1%, −369K shares) — first time this cycle. The persistent positioning dealers carry overnight is short gamma, not long gamma.
What this means: if SPY opens down, dealers must SELL more to hedge — amplifying the move. If SPY opens up, dealers must BUY — also amplifying. Negative gamma turns dealers from shock absorbers into shock amplifiers in both directions.
Volume gamma (today's flow) is still positive (+$44.8B). If institutional flow continues buying on a positive catalyst, the OI structure can repair quickly. The key is getting real buying volume, not just a headlines pop.
